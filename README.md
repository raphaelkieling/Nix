# Nix

Nix is a library for create APIs fast. You need... a unique file if you want.

### Installing

`npm install -g nix`

### Getting start

#### Create a file
```json
// file.json
{
  "person":{
    "name":"Kieling"
  }
}
```

#### Run
```sh
nix --port 300
```

Go to http://localhost:PORT/NAME_FILE
